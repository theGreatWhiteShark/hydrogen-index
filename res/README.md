This folder contains couple of resources to help both development and unit
testing.

### references-index.json

The primary, hand-written version of `hydrogen-index`'s `index.json` used during
development.

### hydrogen-artifacts

Example artifacts of the latest version as well as several historical ones of
different format.

Please ensure to update the resources in sync with
https://github.com/hydrogen-music/hydrogen.

### xsd

Formal definitions of `hydrogen`'s artifacts (or of the subset of those for
which we support XSD).

Please ensure to update the resources in sync with
https://github.com/hydrogen-music/hydrogen.
